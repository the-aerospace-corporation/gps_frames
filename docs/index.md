# GPS Frames

Welcome to the documentation for `gps_frames`.

This library defines reference frames representations and transformations for use in GPS and navigation systems.

## Installation

```bash
pip install gps-frames
```

## Features

- **Reference Frames**: ECEF, ENU, NED, Geodetic (LLA/HAE), ECI.
- **Transformations**: Coordinate transformations between frames.
- **position, Velocity, Vectors**: Classes to handle physical quantities in different frames.
